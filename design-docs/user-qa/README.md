# User QA

Record user decisions and open product questions here.

- `resolved-apple-gateway-defaults.md` — design defaults chosen while
  adapting the mail-gateway baseline (binaries, GraphQL runtime, per-domain
  mechanisms, permission policy).
- `pending-apple-gateway-questions.md` — open questions with proposed
  defaults (timers, Notes fast-read, system-wide notification clearing,
  mail live actions, server mode).
- `qa-example.md` / `pending-example.md` — format templates.
